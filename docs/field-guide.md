# GeoMark — Field Guide

> ⚠️ This guide will be completed as the software matures.

## Setup Sequence

1. **Plant the base** on a tripod with a clear view of the sky
2. **Power on the base unit** — it will begin survey-in automatically
3. **Power on the rover unit** once the base radio link is confirmed
4. **Connect to the rover's WiFi hotspot** from a phone or tablet
5. **Open the field UI** at `http://192.168.4.1:8080`
6. **Wait for RTK Fixed** status before collecting points
7. **Collect points** using the UI
8. **Export data** as CSV or GeoJSON at the end of the session
