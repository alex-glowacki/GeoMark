# GeoMark point code list
# Format: CODE,Description
# One entry per line. Lines starting with # are comments.
# Edit this file and copy to /media/usb/geomark/point_codes.txt on your USB stick.
# Codes are automatically upper-cased on load.
#
# Source: ACG_TOPO_V1.xlsx (KLJ_TOPO_V4_Beta), point codes + line codes
# sections. Action codes (!, JTP, PC, PT, +, -) are intentionally excluded
# -- "+"/"-" are GeoMark's own breakline start/end prefix convention
# (see collector/breaklines.h), not separate point codes; !/JTP/PC/PT are
# not yet implemented in GeoMark.
